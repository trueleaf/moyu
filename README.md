## 简介

**摸鱼**是一款规范研发流程的工具集。规范的软件开发流程将大大提高开发效率。传统软件开发过程中你可能制定了若干`规范`，但实际落地过程中人为去监督这些规范的落地将变得非常困难，**摸鱼**的目的就是来监督这些规范的落地。

## 功能

- 接口文档管理
- 用户权限管理
- 敬请期待
